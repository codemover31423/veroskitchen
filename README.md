# Vero's Kitchen

Static creator/link site for Vero's Kitchen.

## Edit Content

- Main text and links: `index.html`
- Visual styling and animations: `styles.css`
- Scroll/reveal behavior: `script.js`
- Original photos: `images/*.jpeg`
- Live optimized photos: `images/optimized-v2/*.webp`
- Social share preview: `images/social-preview.jpg`

## Before Deploy

Live URL:

```txt
https://codemover31423.github.io/veroskitchen/
```

The sitemap, robots file, and social preview metadata are configured for this URL.

## Free Hosting Options

- GitHub Pages
- Cloudflare Pages
- Netlify

Upload or deploy the contents of this folder as the site root.

## Maintenance

To replace a photo:

1. Put the new original image in `images/`.
2. Keep a clear filename.
3. Ask Codex to optimize it into `images/optimized-v2/`.
4. Update the matching `img src` in `index.html` if the filename changed.

No backend, database, accounts, or payments are handled by this site. PayPal, TikTok, and YouTube are external links.
